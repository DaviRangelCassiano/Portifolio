YouTube sem distrações é um site que reúne vídeos do YouTube sem mostrar recomendações, comentários ou distrações visuais, com o objetivo de permitir que o usuário assista, aprenda e relaxe em um ambiente limpo e focado. 
O projeto é composto pelos arquivos index.html (estrutura principal da página), style.css (estilos e modo noturno), script.js (responsável por carregar e exibir os vídeos dinamicamente), dados.json (lista atualizada dos vídeos obtidos da API) e backend.py (script em Python que usa a API do YouTube Data v3 para buscar os vídeos mais recentes e gerar o JSON). 
Para funcionar, basta instalar o pacote google-api-python-client, inserir a chave da API do YouTube no backend.py, executar o script para gerar o dados.json e depois abrir o index.html no navegador (ou rodar python -m http.server e acessar http://localhost:8000). 
O site mostra os canais em blocos expansíveis, permite abrir vídeos em um player centralizado e oferece modo claro/escuro com preferência salva. É feito em HTML, CSS, JavaScript e Python, totalmente voltado para estudo e foco, sem distrações.
